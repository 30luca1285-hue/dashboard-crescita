# Crescita 🌱

Dashboard di crescita per bambini piccoli, con attività quotidiane, traguardi di sviluppo,
consigli per l'ambiente di casa, diario con foto, monitoraggio di crescita/salute e suoni per la nanna.
Integra tre approcci educativi: **Reggio Emilia**, **Montessori** e **Pikler**.

I contenuti (attività, traguardi, ambiente) si adattano automaticamente all'età del bambino.

## Come si usa

Apri il link, al primo avvio inserisci **nome** e **data di nascita** del bambino, e premi *Inizia*.
Su iPhone/Android puoi aggiungerla alla schermata Home (condividi → "Aggiungi a Home") e usarla come un'app.

## Privacy

È un'app completamente autonoma: **tutti i dati (diario, foto, misurazioni) restano nel tuo telefono**
(localStorage + IndexedDB del browser). Nessun server, nessun account, niente viene caricato online.
Per cambiare nome o data di nascita: icona ⚙️ in alto a destra.
